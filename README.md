# todoada

This project is a To Do application developed to simplify users’ task management. It supports CRUD (Create, Read, Update, Delete) operations. Users can create, list, edit, and delete tasks.  The application has been designed with modern interface design and user experience principles in mind. Its goal is to provide a simple and effective task management system.
